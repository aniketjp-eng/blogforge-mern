# 📝 BlogForge
A modern **AI-powered blogging platform** built with the **MERN Stack**. BlogForge allows users to create, edit, publish, and manage blogs with a clean interface and AI-assisted content generation.

## 🚀 Features
* 📝 Create, edit, and delete blog posts
* 🤖 AI-powered content enhancement (Google Gemini)
* 🖼️ Image upload and optimization
* 💬 Comment system
* 🔐 Secure authentication using JWT
* 📱 Fully responsive design
* ⚡ Fast frontend built with Vite

## 🛠️ Tech Stack
| Frontend                  | Backend             | Database | Other                           |
| ------------------------- | ------------------- | -------- | ------------------------------- |
| React, Vite, Tailwind CSS | Node.js, Express.js | MongoDB  | JWT, Google Gemini AI, ImageKit |

## 📂 Project Structure

```text
BlogForge-MERN/
├── client/      # React Frontend
└── server/      # Express Backend
```

## ⚙️ Installation

### 1. Clone the Repository

```bash
git clone https://github.com/aniketjp-eng/BlogForge-MERN.git
cd BlogForge-MERN
```

### 2. Install Dependencies

#### Backend

```bash
cd server
npm install
npm run server
```

#### Frontend

```bash
cd client
npm install
npm run dev
```

> **Note:** Create a `.env` file inside the `server` folder and add your environment variables before starting the application.
